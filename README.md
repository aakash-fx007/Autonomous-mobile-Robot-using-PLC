# Autonomous-mobile-Robot-using-PLC
Industrial-grade AMR built on Jetson Orin Nano with Delta PLC control, industrial servos, SICK sensors, and SLAMTEC M2M2 LIDAR. Features 120kg chassis with 150kg payload capacity, Modbus communication, advanced SLAM navigation, and real-time obstacle avoidance for heavy-duty material handling in manufacturing and warehouse environments.
